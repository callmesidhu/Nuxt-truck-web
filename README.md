# Nuxt The Full-Stack Vue Framework

A Nuxt.js project powered by ⚡ pnpm for fast and efficient development.

---

## 📦 Repository

GitHub: https://github.com/callmesidhu/Nuxt-truck-web.git

---

## 🧰 Requirements

Make sure you have these installed:

- Node.js (>= 18 recommended)
- pnpm

---

## ⚡ Install pnpm (if not installed)

### Windows / macOS / Linux

npm install -g pnpm

📥 Clone the Project
git clone https://github.com/callmesidhu/Nuxt-truck-web.git

cd Nuxt-truck-web

📦 Install Dependencies
pnpm install

🚀 Run Development Server
pnpm dev

Now open your browser:

👉 http://localhost:3000
